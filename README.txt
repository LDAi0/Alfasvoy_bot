Здравствуйте, код для бота готов и отлично функцанирует, осталось только загрузить его на хостинг и сделать не сколько вещей
1. Добавьте в корневую директорию(там где находиться main.py) два файла формата .png с именами start.png и reg.png
start.png - это будет та фотография которая появляется с сообщением когда пользователь вводит команду /start
reg.png - это будет та фотография когда пользователь нажмет кнопку регистрация
(в боте запрещен скачивание фотографий и я не смог найти такие же, поэтому вы можете добавить их самостоятельно, а пока
там стоят заглушки, вы должны удалить их если хотите поменять фотографии)
2. Введите в файл token.txt токен от бота

И потом зальете его на хостинг, если что тут документация https://beget.com/ru/kb/manual

Деплой бота на сервисе Beget включает в себя несколько шагов. Вот пошаговая инструкция:
1. Зарегистрируйтесь на Beget и создайте аккаунт.
2. Зайдите в панель управления Beget и создайте новый виртуальный хостинг.
3. Загрузите свой код бота на сервер Beget. Вы можете использовать FTP-клиент, встроенный веб-интерфейс или SSH-клиент для этого.
4. Установите необходимые зависимости, как описано в документации библиотеки, которую вы используете для создания бота. Например, если вы используете python-библиотеку telebot, выполните команду `pip install pyTelegramBotAPI`.
5. Создайте файл с токеном вашего бота. Можете использовать метод, описанный в предыдущем ответе, чтобы хранить токен в текстовом файле.
6. Настройте веб-сервер (например, Apache или Nginx) для обработки веб-хуков Telegram API. Для этого вам понадобится настроить виртуальный хост в соответствии с требованиями Telegram.
7. Укажите в настройках бота URL-адрес вашего веб-сервера (с настроенным веб-хуком).
8. После этого ваш бот должен быть доступен и готов к использованию.
Пожалуйста, также обратитесь к документации Beget и Telegram Bot API для получения более подробной информации о настройке сервера и бота.

В файле requirements.txt необходимая библиотека для работы бота
